Pseudo Class :

User Interaction ব্যবহারকারীর কাজ অনুযায়ী

:hover উপরে মাউস নিলে .btn:hover { color: red; }

:active ক্লিক করলে a:active { background: green; }

:focus ইনপুটে কার্সর গেলে input:focus { border: 2px solid blue; }

:visited আগে ক্লিক করা লিংক a:visited { color: purple; }

:checked চেকবক্স টিক চিহ্ন দিলে input:checked { background: yellow; }

:disabled ইনপুট বন্ধ থাকলে button:disabled { opacity: 0.5; }

Structural (অবস্থান ভিত্তিক)

:first-child প্রথম child li:first-child { color: red; }

:last-child শেষ child li:last-child { color: blue; }

:nth-child(n) নির্দিষ্ট child li:nth-child(3) { font-weight: bold; }

:nth-of-type(n) নির্দিষ্ট টাইপের child p:nth-of-type(2)

:not(selector) বাদ দেওয়া p:not(.highlight) { color: gray; }

:empty ফাঁকা উপাদান div:empty { display: none; }

:only-child একমাত্র child হলে p:only-child { color: green; }

Form-related (ফর্ম ইনপুট ভিত্তিক)

:enabled যেটা চালু input:enabled { background: white; }

:disabled যেটা বন্ধ input:disabled { background: gray; }

:required আবশ্যিক ইনপুট input:required { border: red; }

:optional ঐচ্ছিক ইনপুট input:optional { border: green; }

:valid সঠিক ইনপুট input:valid { border: green; }

:invalid ভুল ইনপুট input:invalid { border: red; }
